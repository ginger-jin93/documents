# 概述

## 产品介绍

互动直播NELiveKit (以下简称NELiveKit)提供了一套简单易用的接口，允许开发者通过调用 SDK提供的API，快速地集成互动直播功能至现有应用中。您可以在直播间开播、和其他主播PK等。
