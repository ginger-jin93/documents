## 错误返回结果
- 当前错误来源分别为SDK 与 Rest API
 - 若SDK调用失败，返回的Code 不为0时， 会同时返回错误信息，包含错误码与详细的错误信息。

## 错误码列表

### SDK 内部错误

| 错误码     | 说明                                                       |
|---------|----------------------------------------------------------|
| 0       | 成功                                                       |
| -1      | 错误                                                       |
